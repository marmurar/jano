Sobre mí
========

Marcos Manuel Muraro es el autor de Jano.

El proyecto nació de una necesidad práctica: evaluar sistemas cuyo comportamiento cambia a lo largo del tiempo requiere algo más que un split aleatorio. Jano busca hacer esas simulaciones explícitas, inspeccionables y más fáciles de comunicar.

Podés contactarme acá:

- LinkedIn: `<https://www.linkedin.com/in/marmurar/>`_

Dirección del proyecto
----------------------

Hoy Jano está enfocado en:

- definiciones claras de partición temporal
- soporte para inputs pandas, NumPy y Polars
- splitting eficiente sobre datasets más grandes
- reporting de simulación útil incluso antes de entrenar modelos
